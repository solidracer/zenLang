# ZENLANG VERSION 1.0.0

## VIEWING THE HELP MENU

You can view the help menu by running the program without options:

```
zen
```

## BUILDING THE EXECUTABLE

You can build the executable by running the Makefile in the directory:

```
make
```

## MAKE TARGETS

- `clean`: **Removes obj and bin directory**  

- `cleanobj`: **Removes the obj directory**
